   - 레드-블랙 트리(Red-Black Tree)

   - 최소 신장 트리(Minimum Spanning Tree)  
     - 크루스칼 알고리즘(Kruskal's Algorithm)  
     - 프림 알고리즘(Prim's Algorithm)
